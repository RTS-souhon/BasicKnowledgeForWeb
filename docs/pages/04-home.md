# TOPページ `/`

## 概要

認証済みスタッフへのウェルカムページ。現在の会期名と各情報ページへのナビゲーションカードを表示する。

---

## アクセス制御

| 条件 | 挙動 |
|---|---|
| access_token 有効 | 表示 |
| auth_token 有効（admin/developer） | 表示 |
| それ以外 | `/access` へリダイレクト（middleware.ts） |

---

## 画面構成

```
┌────────────────────────────────────────────┐
│ ヘッダー: [イベント名] [ダッシュボード] [ログアウト]│
├────────────────────────────────────────────┤
│  ようこそ、○○イベントへ                       │
│                                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│  │タイムテーブル│ │  部屋割り  │ │  企画一覧  │  │
│  └──────────┘ └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐               │
│  │ 販売物一覧 │ │  情報検索  │               │
│  └──────────┘ └──────────┘               │

```

### スマートフォン

```
┌──────────────────────┐
│ [イベント名]      [≡] │  ← ハンバーガーメニュー
├──────────────────────┤
│  ようこそ、○○イベントへ │
│                      │
│  ┌────────┐ ┌──────┐ │
│  │タイムテーブル│ │部屋割り│ │  ← 2カラムグリッド
│  └────────┘ └──────┘ │
│  ┌────────┐ ┌──────┐ │
│  │ 企画一覧 │ │販売物一覧│ │
│  └────────┘ └──────┘ │
│  ┌────────┐          │
│  │ 情報検索 │          │
│  └────────┘          │
└──────────────────────┘

[≡] タップ時のドロワーメニュー:
┌──────────────────────┐
│ × 閉じる             │
│ ─────────────────── │
│ ダッシュボード          │
│ ログアウト             │
│ （admin/dev のみ）     │
│   会期切り替え ▼       │
└──────────────────────┘
```

**スマートフォン固有の要件:**
- ナビゲーションカード: 2カラムグリッド（`grid-cols-2`）
- ヘッダーはイベント名とハンバーガーアイコンのみ表示
- ダッシュボードリンク・ログアウト・会期セレクターはドロワーに収納
- ドロワーは画面右からスライドイン（Radix UI Sheet など）
- 各カードのタッチターゲットは最低 44×44px 確保

---

## データ

- 表示する会期名:
  - User: `access_token` Cookie の `event_id` から `access_codes` を取得
  - Admin/Developer: URL クエリパラメータ `?event_id=xxx` で選択した会期を表示（未選択時は空欄）

---

## フロントエンド実装

- `(authenticated)/layout.tsx` に共通ヘッダー・ナビを定義
- `page.tsx` は Server Component
- 会期名の取得:
  - User: `access_token` Cookie の `event_id` を使って Server Component で fetch
  - Admin/Developer: URL の `event_id` パラメータを使用

### `(authenticated)/layout.tsx` の責務

- ヘッダー（イベント名・ダッシュボードリンク・ログアウトボタン）
- ナビゲーション（各コンテンツページへのリンク）
- ロールが admin/developer の場合:
  - 「管理」リンクを追加表示
  - 会期セレクター（`GET /api/access-codes` の一覧からドロップダウン表示）
  - 選択変更で `?event_id=xxx` を URL に付与し、ページ全体のデータを切り替え
- スマートフォン対応:
  - `md:` ブレークポイント未満でハンバーガーメニューに切り替え
  - ドロワーコンポーネントでナビゲーション項目を表示
